# Acoustic guitar note samples — FluidR3 set

Per-note MP3s (E2–Gb5) of the FluidR3_GM SoundFont's Acoustic Guitar (steel)
preset, taken from the pre-rendered set at
https://github.com/gleitz/midi-js-soundfonts (FluidR3 is distributed there
under Creative Commons Attribution 3.0).

Alternate guitar sample set, selectable in the Player's Mixer panel.
