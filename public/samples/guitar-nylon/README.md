# Acoustic guitar note samples — nylon set

Per-note MP3s (E2–Gb5) of the FatBoy SoundFont's Acoustic Guitar (nylon)
preset, taken from the pre-rendered set at
https://github.com/gleitz/midi-js-soundfonts (FatBoy is distributed there
under Creative Commons Attribution Share-Alike 3.0). The bossa guitar.

Alternate guitar sample set, selectable in the Player's Mixer panel; the
Bossa Nova genre presets it.
