# Electric guitar note samples — muted set

Per-note MP3s (E2–Gb5) of the FatBoy SoundFont's Electric Guitar (muted)
preset, taken from the pre-rendered set at
https://github.com/gleitz/midi-js-soundfonts (FatBoy is distributed there
under Creative Commons Attribution Share-Alike 3.0). The palm-muted chick —
made for funk scratching.

Alternate guitar sample set, selectable in the Player's Mixer panel.
